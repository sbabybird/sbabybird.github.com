й•ҝж–Ү

еҹәдәҺ Github Pages жңҚеҠЎжҗӯе»әдёӘдәәеҚҡе®ў

еҹәдәҺ Github Pages жңҚеҠЎжҗӯе»әдёӘдәәеҚҡе®ў иҝҷжҳҜжҲ‘иҮӘе·ұеҶҷзҡ„дёҖзҜҮж–Үз« пјҢеҸ‘еёғеңЁжҲ‘зҡ„еҚҡе®ўгҖӮ

жҗӯе»әзӢ¬з«ӢеҚҡе®ўзҪ‘з«ҷзҡ„ж–№жі•еӨҡз§ҚеӨҡж ·пјҢеёёи§Ғзҡ„ж–№жі•еҢ…жӢ¬з§ҹз”ЁжңҚеҠЎеҷЁиҮӘиЎҢжһ¶и®ҫпјҢжҲ–иҖ…йҖүжӢ©жүҳз®ЎжңҚеҠЎе•ҶжҸҗдҫӣзҡ„еҚҡе®ўеҸ‘еёғжңҚеҠЎгҖӮ

дҪҶжҳҜдёҠиҝ°дёӨз§Қж–№ејҸйғҪдјҡдә§з”ҹдёҖе®ҡзҡ„иҙ№з”ЁпјҢеҜ№дәҺйқһзӣҲеҲ©зҪ‘з«ҷиҖҢиЁҖпјҢдёҚеӨӘеҸӢеҘҪгҖӮ

з»ҸиҝҮеӨҡе№ҙзҡ„е®һи·өпјҢзҺ°еңЁе·Іжңүи¶ҠжқҘи¶ҠеӨҡзҡ„дәәйҖүжӢ©дәҶдҪҝз”Ёе…Қиҙ№дё”жҳ“з”Ёзҡ„ GitHub Pages жңҚеҠЎжқҘжҗӯе»әдёӘдәәеҚҡе®ўгҖӮ

з„¶иҖҢпјҢзҺ°жңүзҡ„дҪҝз”Ёж•ҷзЁӢеӨ§еӨҡдҫ§йҮҚдәҺж“ҚдҪңжӯҘйӘӨпјҢеҜ№дәҺж–°жүӢжҲ–йқһдё“дёҡдәәеЈ«иҖҢиЁҖпјҢйҡҫд»ҘзңҹжӯЈзҗҶи§Је…¶дёӯеҺҹзҗҶгҖӮ

д»–д»¬еҸҜиғҪзңӢеҲ°еҲ«дәәзҡ„еҚҡе®ўж—ўжјӮдә®еҸҲе®һз”ЁпјҢеҚҙеңЁиҮӘе·ұе°қиҜ•жҗӯе»әж—¶еұЎеұЎйҒҮеҲ°жҢ«жҠҳпјҢйҖҗжёҗеӨұеҺ»дҝЎеҝғгҖӮ

иҝҷж— з–‘дјҡжү“еҮ»йӮЈдәӣеёҢжңӣйҖҡиҝҮеҚҡе®ўжҢҒз»ӯеҲӣдҪңж–Үеӯ—гҖҒе№¶дё”еёҢжңӣиғҪеӨҹе…Қиҙ№е»әз«ӢдёӘжҖ§еҢ–з«ҷзӮ№зҡ„дәәзҡ„з§ҜжһҒжҖ§гҖӮ

жң¬ж–Үе°Ҷд»Ӣз»ҚеҰӮдҪ•дҪҝз”Ё GitHub Pages жңҚеҠЎжҗӯе»әе…Қиҙ№зҡ„дёӘдәәеҚҡе®ўзҪ‘з«ҷпјҢеҸҠе…¶иғҢеҗҺзҡ„е®һзҺ°еҺҹзҗҶпјҢд»Ҙеё®еҠ©еҲқеӯҰиҖ…йЎәеҲ©е»әз«ӢиҮӘе·ұзҗҶжғізҡ„еҚҡе®ўз«ҷзӮ№гҖӮ

дәәзҡ„еӨ§и„‘дёҚжҳҜз”ұзҲ¬иЎҢи„‘гҖҒе“әд№іеҠЁзү©и„‘гҖҒжҷәдәәи„‘зҙҜеҠ иҖҢжҲҗ

еҝғзҗҶеӯҰз•ҢжңүдёҖдёӘе…ідәҺдәәзұ»еӨ§и„‘жөҒдј еҫҲе№ҝзҡ„зҗҶи®әпјҡдәәзҡ„еӨ§и„‘дё»иҰҒз”ұдёүеұӮдёҚеҗҢиғҪеҠӣзҡ„еӨ§и„‘жһ„жҲҗпјҢдҪҚдәҺжңҖж ёеҝғзҡ„вҖңзҲ¬иЎҢеҠЁзү©и„‘вҖқгҖҒйҡҸзқҖжј”еҢ–йҖҗжёҗеўһеҠ зҡ„вҖңдҪҺзә§е“әд№іеҠЁзү©и„‘вҖқпјҢд»ҘеҸҠеёҰжқҘжӣҙеӨҡзҒөжҙ»жҖ§е’ҢиҜӯиЁҖиғҪеҠӣзҡ„вҖңй«ҳзә§и„‘вҖқгҖӮ

иҝҷз§ҚзҗҶи®әз»ҷдәәзҡ„ж„ҹи§үе°ұеҘҪеғҸдәәзҡ„еӨ§и„‘жҳҜеғҸжҙӢи‘ұйӮЈж ·пјҢз”ұдёҖдёӘвҖңж—§вҖқзҡ„еӨ§и„‘з»“жһ„йҖҗжёҗж·»еҠ вҖңж–°вҖқеӨ§и„‘еҪўжҲҗгҖӮ

иҖҢдё”жҚўеҸҘиҜқиҜҙпјҢйқ’иӣҷе’Ңйұјд»Қз„¶еңЁжҲ‘д»¬иә«дҪ“йҮҢпјҢжҲ‘д»¬еҸӘжҳҜжңүдәҶжӣҙй«ҳзә§зҡ„дёңиҘҝжҠҠе®ғд»¬еҢ…иЈ№дәҶиө·жқҘгҖӮ

з„¶иҖҢд»ҘдёҠзҗҶи®әеҸӘжҳҜеҝғзҗҶеӯҰе®¶жүҖжҸҗеҮәзҡ„дёҖз§Қй”ҷиҜҜи§ӮзӮ№пјҢиҝ‘е№ҙжқҘйҡҸзқҖз”ҹзү©з§‘еӯҰзҡ„еҸ‘еұ•пјҢи¶ҠжқҘи¶ҠеӨҡзҡ„з§‘еӯҰе®¶ејҖе§ӢжҢҮеҮәиҝҷз§Қй”ҷиҜҜгҖӮ

иҝҷзҜҮгҖҠдҪ зҡ„еӨ§и„‘дёҚжҳҜжҙӢи‘ұпјҢдҪ зҡ„и„‘еӯҗйҮҢе№¶жІЎжңүдёҖдёӘзҲ¬иЎҢеҠЁзү©гҖӢжҳҜз”ұзҘһз»Ҹз§‘еӯҰе®¶ж’°еҶҷпјҢйҮҚзӮ№жҫ„жё…дәҶдёҠиҝ°е…ідәҺвҖңдёүеұӮеӨ§и„‘вҖқзҡ„дј иЁҖгҖӮ

ж–Үз« жҢҮеҮәпјҡд№ӢжүҖд»Ҙиҝҷз§ҚзҗҶи®әжөҒдј з”ҡе№ҝпјҢжҳҜеӣ дёәеӨ§е®¶жҷ®йҒҚеҜ№вҖңиҝӣеҢ–и®әвҖқзҡ„иҝӣеҢ–иҝҮзЁӢжңүжүҖиҜҜи§ЈпјҢеӨ§еӨҡж•°дәәи®ӨдёәиҝӣеҢ–иҝҮзЁӢжҳҜдёҖз§Қе…ҲдҪҺзә§еҶҚйҖҗжёҗй«ҳзә§зҡ„зәҝжҖ§иҝҮзЁӢгҖӮз„¶иҖҢдәӢе®һзҡ„иҝӣеҢ–жҳҜдёҖз§ҚеҗҢжӯҘиҫҗе°„иҝӣзЁӢгҖӮ

дҪҝз”Ё Javascript ж“ҚдҪңз…§зүҮзҡ„ EXIF дҝЎжҒҜ

дҪҝз”Ё Javascript ж“ҚдҪңз…§зүҮзҡ„ EXIF дҝЎжҒҜ еҸҜдәӨжҚўеӣҫеғҸж–Үд»¶ж јејҸпјҲEXIFпјүжҳҜдёҖз§Қи§„иҢғпјҢз”ЁдәҺжҢҮе®ҡеӣҫеғҸе’ҢеЈ°йҹізҡ„ж јејҸгҖӮе®ғйҖҡиҝҮе…ғж•°жҚ®еӯҳеӮЁжҠҖжңҜз»ҶиҠӮпјҢжүҖи°“е…ғж•°жҚ®еҚіз”ЁжқҘжҸҸиҝ°е…¶д»–ж•°жҚ®зҡ„ж•°жҚ®пјҢдҫӢеҰӮзӣёжңәеҲ¶йҖ е•Ҷе’ҢеһӢеҸ·гҖҒеӣҫеғҸжӢҚж‘„ж—Ҙжңҹе’Ңж—¶й—ҙзӯүдҝЎжҒҜгҖӮ

жңҖеҲқпјҢEXIFд»…з”ЁдәҺJPEGе’ҢTIFFдёӨз§ҚеӣҫеғҸж јејҸгҖӮз„¶иҖҢпјҢеҰӮд»ҠпјҢе…¶д»–ж–Үд»¶ж јејҸеҰӮPNGгҖҒWEBPжҲ–HEICд№ҹж”ҜжҢҒEXIFд»Ҙз”ЁдәҺе…ғж•°жҚ®гҖӮ

жң¬ж–Үе°ҶйҮҚзӮ№е…іжіЁJPEGж јејҸпјҢжҺўз©¶е…¶еҶ…йғЁз»“жһ„пјҢз„¶еҗҺеұ•зӨәеҰӮдҪ•еңЁжөҸи§ҲеҷЁзҺҜеўғдёӯйҖҡиҝҮJavaScriptиҜ»еҸ–е’Ңжӣҙж–°зӣёе…ізҡ„е…ғж•°жҚ®гҖӮ

CSS зҡ„иҮӘе®ҡд№үзј“еҠЁжӣІзәҝ

дҪҝз”Ё linearпјҲпјү еҮҪж•°е®һзҺ°жғҠдәәзҡ„ CSS зј“еҠЁеҠЁз”»пјҢйҡҸзқҖж–°зҡ„ CSS linear() зј“еҠЁеҠҹиғҪзҡ„еҮәзҺ°пјҢжҲ‘д»¬еҸҜд»ҘеҲӣе»әиҮӘз„¶ж„ҹи§үзҡ„еҠЁз”»е’ҢиҝҮжёЎзҡ„еҸҜиғҪжҖ§еӨ§еӨ§жү©еұ•гҖӮJhey Tompkins еңЁжң¬ж–Үдёӯд»Ӣз»ҚдәҶ CSS зј“еҠЁзҡ„еҪ“еүҚзҠ¶жҖҒпјҢе№¶жј”зӨәдәҶ linear() еҮҪж•°д»Өдәәжңҹеҫ…зҡ„зү№жҖ§гҖӮ вҖӢвҖӢвҖӢ

йҖҡиҝҮж”ҫеӨ§ ROM иҠҜзүҮжҗһз ҙи§Ј

gbromпјҢйҖҡиҝҮжӢҚж‘„ ROM иҠҜзүҮзҡ„жҳҫеҫ®з…§зүҮпјҢе°ҶйҮҢйқўеӣәд»¶зҡ„дәҢиҝӣеҲ¶д»Јз ҒиҝҳеҺҹдәҶеҮәжқҘпјҢзЎ¬з ҙи§Јзҡ„дё–з•Ңе°ұжҳҜиҝҷд№Ҳжңҙе®һж— еҚҺгҖӮ

free-for.devгҖӮиҝҷдёӘзҪ‘з«ҷзӯӣйҖүж”¶еҪ•дәҶеӨ§йҮҸйқўеҗ‘ејҖеҸ‘иҖ…жҸҗдҫӣзҡ„еңЁзәҝжңҚеҠЎгҖӮж”¶еҪ•зҡ„ж ҮеҮҶйқһеёёдёҘиӢӣпјҡеҝ…йЎ»зңҹжӯЈе…Қиҙ№пјҲе…Қиҙ№иҜ•з”Ёзҡ„дёҚз®—пјүпјҢеҰӮжһңеҘ—йӨҗжңүж—¶й—ҙйҷҗеҲ¶пјҢеҲҷиҮіе°‘жҸҗдҫӣ 1 е№ҙзҡ„е…Қиҙ№ж—¶й—ҙгҖӮж”¶еҪ•зҡ„жңҚеҠЎж¶өзӣ–пјҡдә‘еҹәзЎҖи®ҫж–ҪгҖҒеҶ…е®№жүҳз®ЎгҖҒи®ҫи®Ўиө„жәҗгҖҒеӯ—дҪ“гҖҒеңЁзәҝ IDE гҖҒдә§е“Ғзӣ‘жҺ§зӯүгҖӮ

вҖңдё“жіЁвҖқеҒҡдәӢзңӢдјјз®ҖеҚ•пјҢеҚҙжҳҜи§ЈеҶіеҗ„з§ҚзІҫзҘһеҶ…иҖ—зҡ„иүҜиҚҜгҖӮ

иҝҷйҮҢдјҡйҡ”дёүеІ”дә”еҲҶдә«жҲ‘зңӢеҲ°зҡ„жңүи¶Јзҡ„еҶ…е®№пјҲдёҚдёҖе®ҡжҳҜжңҖж–°зҡ„пјҢдҪҶжҳҜжңүж„ҸжҖқпјүпјҢеӣ дёәеӨ§йғЁеҲҶйғҪдёҺжңәеҷЁжңүе…іпјҢжүҖд»Ҙе…ҲеҸ«е®ғвҖңжңәеҷЁж–Үж‘ҳвҖқеҗ§гҖӮ

е–ңж¬ўзҡ„жңӢеҸӢеҸҜд»Ҙи®ўйҳ…е…іжіЁпјҡ

  • йҖҡиҝҮеҫ®дҝЎе…¬дј—еҸ·вҖңд»Һе®№ең°зӢӮеҘ”вҖқи®ўйҳ…гҖӮ

  • йҖҡиҝҮз«№зҷҪиҝӣиЎҢйӮ®д»¶гҖҒеҫ®дҝЎе°ҸзЁӢеәҸи®ўйҳ…гҖӮ
