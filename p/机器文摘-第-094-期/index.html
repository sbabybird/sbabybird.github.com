йҳ…иҜ»ж—¶й•ҝ: 3 еҲҶй’ҹ

жңәеҷЁж–Үж‘ҳ з¬¬ 094 жңҹ

й•ҝж–Ү

з”Ё C йҮҚеҶҷдёҖдёӘ 2D жёёжҲҸеј•ж“Һ

high_impactпјҢжҳҜдёҖдёӘз”Ё C еҶҷзҡ„ 2D еҠЁдҪңжёёжҲҸеј•ж“ҺпјҢеҫҲз®ҖжҙҒдҪҶжҳҜеҫҲејәеӨ§гҖӮ

дҪңиҖ…еңЁиҝҷзҜҮж–Үз« йҮҢи®°иҝ°дәҶиҝҷдёӘеј•ж“Һзҡ„иҜһз”ҹеҺҶзЁӢгҖӮ

дёҺе…¶д»–жёёжҲҸеј•ж“ҺдёҚеҗҢзҡ„жҳҜпјҢиҝҷдёӘеј•ж“ҺжҳҜе…ҲжңүдәҶ JavaScript зүҲжң¬пјҢз„¶еҗҺжүҚеҒҡзҡ„ C иҜӯиЁҖзүҲжң¬гҖӮ

дҪңиҖ…еңЁж–ҮдёӯдёҚдҪҶеӣһйЎҫдәҶ JavaScript зүҲжң¬зҡ„ејҖеҸ‘е§Ӣжң«пјҢиҝҳиҜҰз»Ҷи®Іи§ЈдәҶ C иҜӯиЁҖзүҲжң¬зҡ„е®һзҺ°жҖқи·ҜпјҢеҢ…жӢ¬пјҡжёІжҹ“гҖҒзў°ж’һгҖҒзІ’еӯҗзі»з»ҹзӯүгҖӮ

жҲ‘еҚҒеҲҶи®ӨеҗҢдҪңиҖ…зҡ„дёҖеҸҘиҜқпјҡвҖңжҲ‘е–ңж¬ўзҡ„жёёжҲҸе’ҢжҠҖжңҜйғҪжңүдёҖдёӘзү№зӮ№пјҢйӮЈе°ұжҳҜд»–д»¬еҫҲе®№жҳ“дёҠжүӢпјҢдҪҶжҳҜдёҚе®№жҳ“зІҫйҖҡгҖӮвҖқпјҲдҪңиҖ…еңЁж–ҮдёӯжҢҮ C иҜӯиЁҖпјү

з»Ҳз«Ҝж“ҚдҪңй«ҳзә§жҠҖе·§

з»ҸеёёеңЁ Linux иҝӣиЎҢжңҚеҠЎз«Ҝзј–зЁӢжҲ–иҖ…иҝҗз»ҙж“ҚдҪңзҡ„дәәе‘ҳпјҢзҰ»дёҚдәҶи·ҹз»Ҳз«ҜгҖҒе‘Ҫд»ӨиЎҢзӘ—еҸЈжү“дәӨйҒ“гҖӮ

иҝҷзҜҮгҖҠз»Ҳз«Ҝж“ҚдҪңй«ҳзә§жҠҖе·§гҖӢж•ҙзҗҶдәҶиӢҘе№ІжқЎйқһеёёжЈ’зҡ„ж“ҚдҪңжҠҖе·§пјҲз§ҳзұҚпјүпјҢеҸҜд»ҘжһҒеӨ§жҸҗеҚҮдҪ зҡ„е·ҘдҪңдҪ“йӘҢгҖӮ

ж‘ҳеҪ•йғЁеҲҶеҰӮдёӢпјҡ

  1. дҪҝз”Ёе‘Ҫд»ӨиЎҢзј–иҫ‘пјҡиҫ“й”ҷй•ҝдёІзҡ„еӨ§ж®өе‘Ҫд»ӨдёҚиҰҒзқҖжҖҘпјҢж— йңҖйҮҚж–°иҫ“е…ҘпјҢCtrl+E еҸҜд»Ҙиҝӣе…Ҙзј–иҫ‘з•ҢйқўпјҢзӣҙжҺҘзј–иҫ‘дҪ зҡ„еӨ§ж®өе‘Ҫд»ӨгҖӮ
  2. йҖҡиҝҮ tmux и„ҡжң¬жҺ§еҲ¶з»Ҳз«ҜиЎҢдёәпјҲеҸҜд»Ҙе®ҢжҲҗдёҖдәӣиҮӘеҠЁеҢ–дәӨдә’ж“ҚдҪңпјүгҖӮ
  3. fzf жЁЎзіҠжҹҘжүҫж•ҙеҗҲеҲ°е‘Ҫд»ӨиЎҢеҺҶеҸІжҹҘжүҫдёӯпјҢCtrl + R еҸҜд»Ҙеҝ«йҖҹжҗңеҜ»жӣҫз»Ҹжү§иЎҢиҝҮзҡ„е‘Ҫд»ӨгҖӮ

и®Ўз®—жңәйҖҶеҗ‘е·ҘзЁӢеӨ§е…Ё

жҸҗеҲ°вҖңйҖҶеҗ‘е·ҘзЁӢвҖқеӨ§е®¶йҰ–е…ҲжғіеҲ°зҡ„е°ұжҳҜвҖңз ҙи§ЈвҖқпјҢдәӢе®һдёҠд№ҹжҳҜиҝҷж ·гҖӮ

дёҚиҝҮйҷӨдәҶжҗһи¶ҠзӢұгҖҒз ҙи§Јд№ӢеӨ–пјҢеӯҰд№ йҖҶеҗ‘е·ҘзЁӢд№ҹжҳҜдәҶи§Ји®Ўз®—жңәеә•еұӮеҺҹзҗҶзҡ„з»қдҪійҖ”еҫ„гҖӮ

гҖҠйҖҶеҗ‘е·ҘзЁӢгҖӢиҝҷжң¬з”өеӯҗд№Ұж•ҙзҗҶдәҶдёҖд»ҪеҚҒеҲҶиҜҰе°Ҫзҡ„еӯҰд№ жүӢеҶҢгҖӮ

еҶ…е®№ж¶өзӣ–дәҶ x86гҖҒarm зӯүдё»жөҒе№іеҸ°е…ідәҺеҰӮдҪ•иҝӣиЎҢвҖңйҖҶеҗ‘вҖқзҡ„е…ЁйғЁе·ҘдҪңжҢҮеҚ—гҖӮ

жүӢеҶҢдёҚд»…д»Ӣз»ҚеҹәзЎҖзҡ„еҺҹзҗҶзҹҘиҜҶпјҢиҝҳж•ҙзҗҶдәҶе…ідәҺзҺҜеўғе®үиЈ…гҖҒй…ҚеҘ—зҡ„е·Ҙе…·дҪҝз”Ёзӯүж–№йқўзҡ„дҝЎжҒҜпјҢйҖӮеҗҲиҫ№еӯҰиҫ№з»ғ

иө„жәҗ

зәҜ Python еҶҷ Web з•Ңйқў

FastHTMLпјҢдёҖдёӘеҘҪзҺ©зҡ„йЎ№зӣ®пјҢз”ЁзәҜ Python ејҖеҸ‘зҺ°д»ЈзҪ‘йЎөеә”з”ЁгҖӮ

дёҚйңҖиҰҒдәҶи§ЈеүҚз«ҜзҹҘиҜҶпјҢеҸӘйңҖиғҪеҶҷ Python д»Јз ҒпјҢеҚіеҸҜеҒҡеҮәзҫҺи§Ӯжҳ“з”Ёзҡ„зҪ‘йЎөзЁӢеәҸгҖӮ вҖӢвҖӢвҖӢ

дёҖдёӘе®һйӘҢжҖ§иҙЁзҡ„ Js еј•ж“Һ

porfforпјҢдёҖдёӘд»Һйӣ¶е®һзҺ°зҡ„ JS и§ЈйҮҠеҷЁгҖӮ

Porffor жҳҜдёҖдёӘзӢ¬зү№зҡ„ JS еј•ж“Һ/зј–иҜ‘еҷЁ/иҝҗиЎҢж—¶пјҢе°Ҷ JS д»Јз ҒжҸҗеүҚзј–иҜ‘дёә WebAssembly жҲ–еҺҹз”ҹд»Јз ҒгҖӮ

зӣ®еүҚиҜҘйЎ№зӣ®еңЁ Github ејҖжәҗпјҢд»…з”ЁдәҺз ”з©¶з”ЁйҖ”гҖӮ

дёҚиҝҮпјҢд»ҺйЎ№зӣ®дё»йЎөз»ҷеҮәзҡ„жөӢиҜ•ж•°жҚ®жқҘзңӢпјҢзӣ®еүҚиҝҳдёҚе…·еӨҮе®һз”ЁжҖ§пјҢд»…йҖҡиҝҮдәҶ 35% зҡ„иҜӯжі•ж ҮеҮҶжөӢиҜ•гҖӮ

еңЁе®№еҷЁйҮҢиҝҗиЎҢ MacOS

Docker-OSXпјҢиҝҷдёӘејҖжәҗйЎ№зӣ®еҸҜд»Ҙи®©дҪ еңЁ docker йҮҢиҝҗиЎҢ MacOS зі»з»ҹгҖӮ

йҖҡиҝҮејәеӨ§зҡ„иҷҡжӢҹжңәжЁЎжӢҹеҷЁ QEMU е®һзҺ°гҖӮ вҖӢвҖӢвҖӢ

з§‘з ”еҶҷдҪңдё“з”Ёзј–иҫ‘еҷЁ

stempadпјҢдёҖдёӘдё“й—Ёз”ЁдәҺз§‘з ”еҶҷдҪңзҡ„еңЁзәҝе·Ҙе…·пјҢжөҸи§ҲеҷЁи®ҝй—®зӣҙжҺҘдҪҝз”ЁпјҢжүҖжңүж•°жҚ®дҝқеӯҳеңЁжң¬ең°гҖӮ

зү№иүІжҳҜеҶ…еөҢеҸҜи§ҶеҢ–е…¬ејҸзј–иҫ‘еҷЁд»ҘеҸҠе…¬ејҸеӣҫеҪўз»ҳеҲ¶еҠҹиғҪгҖӮ вҖӢвҖӢвҖӢ

и®ўйҳ…

иҝҷйҮҢдјҡдёҚе®ҡжңҹеҲҶдә«жҲ‘зңӢеҲ°зҡ„жңүи¶Јзҡ„еҶ…е®№пјҲдёҚдёҖе®ҡжҳҜжңҖж–°зҡ„пјҢдҪҶжҳҜжңүж„ҸжҖқпјүпјҢеӣ дёәеӨ§йғЁеҲҶйғҪдёҺжңәеҷЁжңүе…іпјҢжүҖд»Ҙе…ҲеҸ«е®ғвҖңжңәеҷЁж–Үж‘ҳвҖқеҗ§гҖӮ
